Welcome to Fabsology Chat App with Word Prediction

This project showcases a chat application implemented in C# using TCP/IP for communication between a server and clients. The chat app comes with a word prediction feature based on frequency analysis.

Features:

  • Chat with multiple clients in real-time
  • Word prediction to suggest the next word while typing
  • Modern user interface and interactive console input

Check out the repository on GitHub for more details and to explore the source code.
